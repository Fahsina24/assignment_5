Important FAQ

What is the Document Object Model (DOM)?

DOM means Document Object Model. It creates object for the document for a webpage. Html makes the basic structure, Css makes the design and javascript makes it effective by using DOM which helps to excess it dynamically.

How do you select an element from the DOM?

5 types of methods are possible for selecting an element or elements by using DOM such as getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll(). For selecting an particular element unique ID can be used by getElementById() methods which can easily able to excess that particular content from the html file.

What is event delegation in the context of the DOM, and why it is useful?

This technique helps to handle multiple element by using a single event listener to its parent element and performing some tasks. It is important because if I use it, I do not need to attach multiple event listeners for particular tasks. Moreover, it can make the code more readable and fewer code lines as well by delegating the html document.

How do you manipulate an element's attributes and styles using the DOM?

For manipulating an elements attributes some methods can be used such as setAttribute(to something in the html file),getAttribute, removeAttribute and so on. While for manipulating styles css style tag can be used while the property need to be set as camelCase for example - if I have a h4 tagged element and I want to make the background color blue using DOM, I need to use document.getElementsByTagName("h4").style.backgroundColor = "blue"
